
# Optimizando la Logística con Dropi

DropCost Master se integra con el flujo de Dropi para darte visibilidad sobre las devoluciones y fletes efectivos.

## Sincronización de Estados
El sistema mapea automáticamente los estados de Dropi a la terminología de DropCost:
- **Entregado**: Cierra el ciclo de costeo positivo.
- **Devolución**: Activa el cálculo de pérdida por flete y costo de producto si aplica.

## Recomendaciones de Despacho
Para mantener un margen saludable, asegúrate de despachar el mismo día si la orden entra antes de las 2 PM. La velocidad de entrega reduce drásticamente tu % de devoluciones.

## DropCredits y Fletes
Recuerda que puedes usar tus **DropCredits** para pagar fletes prepagados y obtener mejores tarifas gracias a nuestras alianzas logísticas.
