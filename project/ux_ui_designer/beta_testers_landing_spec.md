# Especificaciones de Diseño: Landing Page Beta Testers

**Destinatario**: UX/UI Designer
**Autor**: Product Manager
**Fecha**: 03-03-2026
**Objetivo**: Crear una interfaz (landing page/graphic) enfocada en la captación de usuarios para el programa Beta de DropCost Master. No se requiere implementación en frontend, el entregable final será un PNG exportado desde Stitch.

## 1. Requisitos Visuales y Estilo
- **Resolución**: Desktop (1440x1024 preferiblemente o similar).
- **Tema**: *Dark mode* (Obligatorio). Usa los fondos oscuros corporativos (`var(--bg-primary)` / `#0A0F1C` o similares).
- **Branding**: Incluir el logo oficial de DropCost Master (o espacio para él) y adherirse a la paleta de colores oficial (azules corporativos `var(--color-primary)` `#0066FF`, toques de verde para success, y grises oscuros).
- **Sensación**: Premium, exclusiva, tecnológica, "invitación a un grupo selecto".

## 2. Estructura Sugerida
Se recomienda un layout tipo "Split" (pantalla dividida) o una Landing Page centrada:
- **Hero Section**: Titular impactante y propuesta de valor (2 meses GRATIS).
- **Beneficios (Checklists)**: Usar íconos llamativos (ej. check verde, regalos, estrellas).
- **Call to Action (CTA)**: Botón principal "Unirme al Programa Beta" (color azul vibrante).
